# Knowledge Graph Embedding Based Question Answering
Knowledge Graph Embedding Based Question Answering, WSDM 2019

## Installation
- Requirements
1. fuzzywuzzy
2. scikit-learn
3. torchtext
4. nltk
5. pytorch
6. numpy
- Usage
1. cd KEQA_WSDM19
2. pip install -r requirements.txt
3. sh main.sh
